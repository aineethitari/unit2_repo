## Code

```.py
from matplotlib import pyplot as plt

n = 100
x = []
y = []
step = 20/100
start = -10
for i in range(n):
    x.append(start)
    y.append(abs(start))
    start+= step


plt.plot(x, y)
plt.show()
```
